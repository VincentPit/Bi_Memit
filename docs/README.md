Welcome to the Bi-MEMIT documentation! This directory contains comprehensive guides, API references, and tutorials for using the Bi-MEMIT library.

## Documentation Structure

- `api/` - Detailed API reference documentation
- `tutorials/` - Step-by-step tutorials and guides
- `examples/` - Code examples and use cases
- `advanced/` - Advanced topics and customization
- `changelog.md` - Version history and changes

## Building Documentation

To build the documentation locally:

```bash
cd docs
pip install -r requirements.txt
make html
```

The documentation will be available in `_build/html/index.html`.
